# Description
An interactive web tool to extract, visualize, and customize CSS styles from any Elementor-based website. Fetch core stylesheets instantly and apply them to standard components to preview and build custom elements visually.

# 🚀 Key FeaturesCSS
- __Extractor:__ Scrapes Elementor Kit, style.css, and engine.css automatically.
- __Component Visualizer:__ Applies extracted styles to buttons, spacers, and headings in real time.
- __Button Builder:__  Visual interface to design custom buttons using the retrieved style classes.
- __Local Persistence:__ Saves all your configurations, palettes, and created buttons in localStorage.
- __Quick Export:__ Copies the final HTML and CSS code snippet with a single click.🛠️ Tech StackHTML5 / CSS3 / JavaScript (ES6+)Fetch API (for stylesheet scraping)LocalStorage API (for client-side data persistence)

## 🛠️ Tech Stack
- HTML5 / CSS3 / JavaScript (ES6+) / React / Vite 
- LocalStorage API (for client-side data persistence)
